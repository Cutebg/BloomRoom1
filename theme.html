Project theme

I chose the flower arrangements theme for creating a website for flower arrangements as the final project of my class, "Interactive Web Page Scripting with Lab." My decision was driven by a mix of personal interests and academic goals. I wanted to apply the skills I learned in this course while exploring a topic that I found engaging and relevant. This project provided an opportunity to apply web development concepts and techniques I was learning in class, including HTML, CSS, and JavaScript, practically and creatively. I designed a website for users to easily browse and select flower arrangements, focusing on intuitive navigation and appealing visuals. This project enhanced my technical skills and confidence in web development, combining classroom learning with practical experience. It was a rewarding experience that contributed to my growth as a student and aspiring web developer.
